# Dial-up Loader

For Birb's Nest #WCCChallenge 2023-08-06, for the theme "Incomplete"

## About
This project was created as a fun and educational exploration into simulating image rendering and the user experience of the early Internet era. Built using p5.js. Relive the days when patience and being in a state of "incompletion" was key in digital media consumption!
